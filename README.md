### Full Text Content
| Deutsch (Original) | English (Translation) |
| :--- | :--- |
| **Liebe Mitarbeitende, unser Büro möchte Ressourcen bewusster nutzen. Bitte halten Sie sich an folgende Richtlinien:** | **Dear employees, our office wants to use resources more consciously. Please adhere to the following guidelines:** |
| **Papierverbrauch:** Bitte verwenden Sie Papier doppelseitig und recyceln Sie ungenutzte Blätter. Notizen sollten auf Schmierpapier geschrieben werden, das bereits auf einer Seite bedruckt ist. Für Kunststoffe, Metalle und andere wiederverwertbare Materialien stehen die orangefarbenen Tonnen bereit. | **Paper consumption:** Please use paper double-sided and recycle unused sheets. Notes should be written on scrap paper that is already printed on one side. Waste paper belongs in the green bin in the entrance area. Orange bins are available for plastics, metals, and other recyclable materials. |
| **Kaffeeküche:** Um Wasser zu sparen, ist das Spülen von Hand nicht mehr gestattet. Nutzen Sie die Geschirrspülmaschine und schalten Sie diese erst ein, wenn sie voll ist – idealerweise einmal täglich. | **Coffee kitchen:** To save water, washing by hand is no longer permitted. Use the dishwasher and only turn it on when it is full – ideally once a day. |
| **Stromsparen:** Bitte schalten Sie Ihren Computer und das Licht aus, wenn Sie den Arbeitsplatz verlassen. Reduzieren Sie die Nutzung von Standby-Modi. Anstatt die Heizung hochzudrehen, nutzen Sie bitte die bereitgestellten Decken und achten Sie darauf, dass Fenster geschlossen sind. | **Saving electricity:** Please turn off your computer and the light when you leave the workplace.
